Variable Usage

Variables defined here will be available in your templates using the {{"{{"}}variable_name{{"}}"}} syntax.

Example

{
    "company_name": "Acme Corp",
    "support_email": "support@acme.com",
    "unsubscribe_url": "https://acme.com/unsubscribe"
}

Variable Priority

  1. Recipient variables - highest priority, from recipient list
  2. Campaign variables - defined on this page
  3. Global variables - defined in Settings
